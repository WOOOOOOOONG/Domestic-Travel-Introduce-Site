# TravelMaker
KH 정보교육원에서 4인의 팀 구성으로 진행한 웹 개발 프로젝트입니다.<br>
해외 여행이 유행인 최근, 사람들이 국내의 아름다운 경관을 먼저 알아줬으면 하는 생각으로 만든 국내 여행지 소개 사이트입니다.
<br>

## 개발 환경
* Programming Language    
Java 7
* DB  
Oracle
* Web  
HTML, CSS, Javascript, jQuery
* Server  
Tomcat 8.5, JSP/Servlet
